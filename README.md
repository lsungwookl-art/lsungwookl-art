## 🚀 주요 프로젝트

- **캐추파이 (Catchupai)** — AI 기반 학습 이탈 방지 플랫폼  
  학습자의 이탈 패턴을 분석하고, AI로 맞춤형 개입을 제공하는 EdTech 서비스

- **invisible AI bridge** — AI 교육기술 인프라  
  AI를 교육 환경에 자연스럽게 녹여내는 브릿지 시스템

- **mes_make** — 제조 실행 시스템(MES)  
  TypeScript 기반의 제조 공정 관리 웹 애플리케이션

- **blog_auto** — 블로그 자동화  
  TypeScript로 구현한 콘텐츠 자동 발행 파이프라인
